For over 6 years now, HasGeek’s focus has been to provide a platform for people in technology meet each other, collaborate on new opportunities and share knowledge and ideas. We’ve done this by organizing conferences, hacknights, workshops and meetups around topics such as front end engineering, big data and cloud computing, JavaScript, Android, and devops

In 2017 we want to explore how we can do the same for game development, and this is our first step towards that.

Who is this for?

If you are able to create art digitally or know basic scripting or programming it would be a great advantage, but completely optional. This workshop will also be useful for those who are interested in participating in the Global Game Jam 2017 but don’t have any experience in making games. If you’ve tried your hand at GameMaker (perhaps by following online tutorials) but hit a wall, then this is the place for you!

What is GameMaker Studio?

GameMaker Studio is an easy to use game creation toolkit for Windows with an intuitive ‘Drag and Drop’ development environment as well as support for advanced features via their scripting interface - GameMaker Language.
More about GameMaker:http://www.yoyogames.com/gamemaker

What will you learn?

  • A primer on GameMaker Studio

  • Getting Started

  • Resource Management

  • Sprites and Objects

  • The GameMaker Language

  • Making a Game

  • Debugging

  • Game Settings and Exporting

About the instructors

Joel Johnson

Joel Johnson has been working in game development for close to fifteen years. He has built games for PC, DTH and mobile platforms.

Yadu Rajiv

Yadu Rajiv is a game designer and developer, as well as a curator and evangelist for independent games. A maverick who designs games and game-like experiences, he also writes code and creates art for games. After studying English Literature, he went to NID, Bangalore for a degree in Design for Digital Experience. Yadu worked as a Senior User Experience Designer before joining Srishti Institute of Art, Design and Technology, where he teaches User Experience Design, Game Design and Game Development.
